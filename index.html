    Responda as perguntas abaixo:

  1. Na tag INPUT com atributo type="text", para que serve o atributo "placeholder"?

    2. R= Usado Para fornecer uma dica rápida ou exemplo do que o usuário deve digitar em um campo de entrada. Isso aparece no campo de texto antes de o usuário digitar qualquer coisa e desaparece quando o usuário começa a digitar

  2. Na tag INPUT com atributo type="radio" ou type="checkbox", para que serve o atributo "checked"?

    3. R=O atributo "checked" em uma tag INPUT com o tipo "radio" ou "checkbox" serve para especificar se o botão de opção (radio button) ou a caixa de seleção (checkbox) devem estar marcados ou não por padrão quando a página é carregada. Quando o atributo "checked" está presente na tag INPUT com o valor "checked", o botão de opção ou a caixa de seleção aparecerá marcado por padrão.

  3. Na tag INPUT para que serve o atributo "value"?

    4. R=O atributo value serve para definir o valor a ser enviado por este item. O atributo checked indica se este item deve estar selecionado por padrão.

  4. Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?

    5. R=Para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez, é utilizado o atributo “name” com o mesmo valor para todos os botões de rádio que você deseja agrupar. Quando você define o mesmo valor para o atributo “name” em vários botões de rádio, eles são tratados como parte do mesmo grupo e apenas um deles pode ser selecionado ao mesmo tempo.

  5. Para que serve a tag LABEL? Para que serve o atributo "for" nessa tag?

    6. R=A tag LABEL é usada para associar um rótulo descritivo a um elemento de formulário, como um campo de texto, uma caixa de seleção ou um botão de rádio.

  6. Na tag OPTION para que serve o atributo "selected"?

    7. R=O atributo SELECTED a tag OPTION é usado em um elemento de uma lista suspensa SELECT em HTML

  7. Nas tags de formulário para que serve o atributo "required"?

    8. R=usado para indicar que um determinando campo de formulário é obrigatório para o envio do mesmo

  8. Para que serve a tag TEXTAREA?

    9. R=Utilize o Textarea quando: Houver a necessidade de inserir dados de forma textual através de um sistema ou aplicação; A informação textual requerida for relativamente longa, exigindo múltiplas linhas de texto.

  9. Na tag FORM para que serve o atributo ACTION?

    10. R=define o local (uma URL) em que os dados recolhidos do formulário devem ser enviados

  10. a tag FORM para que serve o atributo "method"?

    11. R=method define qual o método HTTP para enviar os dados (ele pode ser "GET" ou "POST" (veja as diferenças aqui).

  11. Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag FORM?

    12. R=Password: permite que o usuário digite um valor que não é exibido de forma legível, sendo utilizado para a entrada de senhas. Nesse caso, o elemento
    deve ter o atributo method definido com o valor post, para que a senha não seja mostrada na linha de endereço.

  12. Qual valor deve ser informado no atributo "method" na tag FORM em um formulário que será usado para fazer upload de arquivos?

    13. R=Existem dois valores possíveis para o atributo "method": "get" e "post".

  13. O que as tags FIELDSET e LEGEND produzem?

    14. R=Essa tag é bem utilizada para definir uma melhor organização dos elementos de formulários dentro dos mesmos. "Legend Field Element" ou legenda é utilizado para rotular, identificar o campo definido FIELDSET FIELDSET de um formulário.
